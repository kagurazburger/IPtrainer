# 局域网访问说明（最简单的方法）

## 如果手机和电脑在同一WiFi网络

这是最简单、最快的方法，无需任何额外配置！

### 步骤1：查看电脑IP地址

在命令行中运行：
```bash
ipconfig
```

找到"无线局域网适配器 WLAN"或"以太网适配器"下的 **IPv4 地址**，例如：
```
IPv4 地址 . . . . . . . . . . . . : 192.168.1.100
```

### 步骤2：配置前端使用局域网IP

创建或编辑 `.env.local` 文件（在项目根目录）：

```
VITE_API_URL=http://192.168.1.100:8000
```

**注意**：将 `192.168.1.100` 替换为你实际的IP地址。

### 步骤3：重启前端

如果前端正在运行，停止它（Ctrl+C），然后重新启动：
```bash
npm run dev
```

### 步骤4：在手机上访问

在手机浏览器中打开：
```
http://192.168.1.100:5173
```

**注意**：将 `192.168.1.100` 替换为你实际的IP地址。

## 注意事项

1. **防火墙**：确保Windows防火墙允许5173和8000端口的访问
2. **同一网络**：手机和电脑必须在同一WiFi网络
3. **IP变化**：如果电脑IP地址变化，需要更新配置

### 如果无法访问，检查防火墙：

1. 打开"Windows Defender 防火墙"
2. 点击"高级设置"
3. 点击"入站规则" -> "新建规则"
4. 选择"端口" -> 下一步
5. 选择"TCP"，输入端口 `5173` 和 `8000`
6. 允许连接 -> 下一步 -> 完成
