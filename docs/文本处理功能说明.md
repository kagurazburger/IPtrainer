# 文本批量处理功能说明

## 功能概述

基于 Gemini AI 的文本批量处理工具，支持上传文本文件（.txt, .srt, .md, .markdown），使用自定义 system prompt 进行 AI 处理，处理后的文件自动保存到 `output` 文件夹。

## 配置要求

### 1. Gemini API Key 配置

需要在以下方式之一配置 Gemini API Key：

**方式 1：环境变量**
```bash
# Windows PowerShell
$env:GEMINI_API_KEY="your-api-key-here"

# Windows CMD
set GEMINI_API_KEY=your-api-key-here

# Linux/Mac
export GEMINI_API_KEY=your-api-key-here
```

**方式 2：config.json**
在 `backend/config.json` 中添加：
```json
{
  "gemini_api_key": "your-api-key-here"
}
```

### 2. 安装后端依赖

```bash
cd backend
pip install -r requirements.txt
```

## 使用方法

1. **启动后端服务**
   ```bash
   cd backend
   python main.py
   ```
   后端默认运行在 `http://localhost:8000`

2. **启动前端服务**
   ```bash
   npm run dev
   ```
   前端默认运行在 `http://localhost:5173`

3. **使用文本处理功能**
   - 在主页点击 "文本处理" 按钮
   - 输入自定义的 system prompt（AI 处理指令）
   - 上传一个或多个文本文件（支持 .txt, .srt, .md, .markdown）
   - 点击 "开始批量处理"
   - 处理完成后，文件会自动保存到项目根目录的 `output` 文件夹中

## 输出文件

处理后的文件会保存在 `output` 文件夹中，文件名格式为：`原文件名_processed.扩展名`

例如：
- 上传 `example.txt` → 输出 `example_processed.txt`
- 上传 `subtitle.srt` → 输出 `subtitle_processed.srt`

## 注意事项

- 确保后端服务正在运行
- 确保已正确配置 Gemini API Key
- 大文件处理可能需要较长时间，请耐心等待
- 处理后的文件会覆盖同名的已存在文件
