# 手机同步使用说明

## 功能概述

现在应用支持云端同步功能，可以在电脑和手机之间同步所有卡牌数据。本地文件夹中的JSON文件也会自动同步到云端。

## 设置步骤

### 1. 在电脑上创建/登录云账户

1. 打开应用
2. 点击右上角的用户图标
3. 在"Cloud Profile"标签页中：
   - **如果已有同步密钥**：在输入框中粘贴你的同步密钥，点击"Restore Profile"
   - **如果没有密钥**：点击"Generate New Profile"生成新的同步密钥
4. 复制并保存你的同步密钥（格式类似：`ABCD-1234-EFGH`）

### 2. 在手机上访问应用

#### 方法一：通过浏览器访问（推荐）

1. 确保手机和电脑在同一网络，或者应用已部署到公网
2. 在手机浏览器中访问应用的URL（例如：`http://你的电脑IP:5173` 或部署后的域名）
3. 点击右上角用户图标
4. 在"Cloud Profile"标签页中，粘贴你的同步密钥
5. 点击"Restore Profile"登录

#### 方法二：安装为PWA（Progressive Web App）

1. 在手机浏览器中打开应用
2. 点击浏览器的"添加到主屏幕"选项（不同浏览器位置不同）
3. 应用会像原生应用一样安装到手机桌面
4. 打开应用后，使用相同的同步密钥登录

### 3. 同步机制

#### 自动同步
- 创建新卡牌组时：自动保存到本地文件夹 + 同步到云端
- 删除卡牌组时：自动同步到云端
- 修改卡牌时：自动同步到云端
- 应用启动时：自动合并本地文件夹和云端的数据

#### 手动同步
- 点击右上角的云图标按钮可以手动触发同步
- 同步会合并本地文件夹和云端的所有卡牌

## 数据同步说明

### 本地文件夹同步
- 应用会自动读取 `decks` 文件夹中的所有JSON文件
- 这些文件会自动同步到云端
- 在手机上，由于无法直接访问本地文件夹，所有数据都通过云端同步

### 云端同步
- 所有卡牌数据存储在云端（kvdb.io）
- 使用同步密钥作为唯一标识
- 数据包括：
  - 所有卡牌组
  - AI配置（Volcano API密钥、模型选择等）
  - 用户偏好设置

## 使用流程

### 在电脑上
1. 将卡牌JSON文件放入 `decks` 文件夹
2. 打开应用，应用会自动加载这些文件
3. 登录云账户后，这些卡牌会自动同步到云端

### 在手机上
1. 打开应用并登录（使用同步密钥）
2. 所有云端卡牌会自动加载
3. 可以正常使用所有功能（学习、复习等）
4. 在手机上创建的新卡牌也会同步到云端和电脑

## 注意事项

1. **同步密钥安全**：请妥善保管你的同步密钥，丢失后无法恢复数据
2. **网络连接**：同步需要网络连接，确保手机和电脑都能访问互联网
3. **数据合并**：如果本地文件夹和云端都有同名卡牌组，会优先使用云端版本
4. **首次同步**：首次登录时，云端数据会覆盖本地数据（如果有冲突）

## 故障排除

### 同步失败
- 检查网络连接
- 确认同步密钥正确
- 尝试手动点击同步按钮

### 数据不一致
- 点击手动同步按钮强制同步
- 检查本地文件夹中的JSON文件是否正确

### 手机无法访问
- 确保应用已部署到公网，或手机和电脑在同一局域网
- 检查防火墙设置
- 尝试使用移动数据网络

## 技术细节

- **云端存储**：使用 kvdb.io 作为后端存储
- **同步协议**：基于HTTP REST API
- **数据格式**：JSON格式，与本地文件格式兼容
- **冲突解决**：以云端数据为准，本地文件夹数据作为补充
