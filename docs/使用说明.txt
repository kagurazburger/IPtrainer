============================================================
  Live Memory Trainer - 使用说明
============================================================

启动
----
双击 start.bat 即可。会自动检查环境、安装依赖、启动前后端。

若后端依赖安装失败，先双击运行一次：
  backend\install_deps.bat
再重新运行 start.bat。

配置
----
编辑 backend\config.json，填写本机的 Whisper、Llama 等路径。

环境要求
----
- Python 3.8+
- Node.js 16+

详见 README_LOCAL.md
