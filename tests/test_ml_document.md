# 机器学习基础概念

## 监督学习 (Supervised Learning)

监督学习是一种机器学习方法，其中算法从标记的训练数据中学习。训练数据包含输入特征和对应的正确输出标签。

### 关键概念

1. **特征 (Features)**: 输入数据的属性或变量
2. **标签 (Labels)**: 我们想要预测的目标值
3. **训练集 (Training Set)**: 用于训练模型的数据
4. **测试集 (Test Set)**: 用于评估模型性能的数据

### 常见算法

- **线性回归 (Linear Regression)**: 预测连续数值
- **逻辑回归 (Logistic Regression)**: 分类问题
- **决策树 (Decision Trees)**: 基于规则的分类和回归
- **支持向量机 (SVM)**: 寻找最佳分离超平面的分类器
- **神经网络 (Neural Networks)**: 模拟人脑神经元连接的复杂模型

### 评估指标

- **准确率 (Accuracy)**: 正确预测的比例
- **精确率 (Precision)**: 真正例占预测为正例的比例
- **召回率 (Recall)**: 真正例占实际正例的比例
- **F1分数**: 精确率和召回率的调和平均值

## 无监督学习 (Unsupervised Learning)

无监督学习不使用标记数据，算法试图从数据中发现隐藏的模式或结构。

### 主要类型

1. **聚类 (Clustering)**: 将相似的数据点分组
2. **降维 (Dimensionality Reduction)**: 减少特征数量同时保留重要信息
3. **关联规则学习**: 发现数据项之间的关系

### 应用场景

- 客户细分
- 异常检测
- 推荐系统
- 数据压缩
