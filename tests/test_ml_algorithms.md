# 机器学习算法导论

## 监督学习算法

### 线性回归 (Linear Regression)
线性回归是一种预测连续数值的监督学习算法。

**数学公式**: `y = wx + b`
- y: 预测值
- x: 输入特征
- w: 权重参数
- b: 偏置项

**应用场景**: 房价预测、销售额预测

### 决策树 (Decision Tree)
决策树通过一系列问题将数据分类或回归。

**优点**:
- 易于理解和解释
- 可以处理数值和分类特征
- 不需要特征缩放

**缺点**:
- 容易过拟合
- 对噪声敏感

## 无监督学习算法

### K-means聚类
K-means将数据点分为k个簇。

**算法步骤**:
1. 随机选择k个中心点
2. 将每个数据点分配到最近的中心
3. 重新计算每个簇的中心
4. 重复步骤2-3直到收敛

**参数选择**: 肘部法则选择k值

## 模型评估指标

### 分类指标
- 准确率 = (TP + TN) / (TP + TN + FP + FN)
- 精确率 = TP / (TP + FP)
- 召回率 = TP / (TP + FN)
- F1分数 = 2 * 精确率 * 召回率 / (精确率 + 召回率)

### 回归指标
- 均方误差 (MSE)
- 平均绝对误差 (MAE)
- R²分数
